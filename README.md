# QUEST.exe 🕹️

A Python-based interactive adventure game built with **Streamlit**. 

## ✨ Features
* **Choice-Based Narrative:** Every decision impacts your journey.
* **Progress Tracking:** Includes a dynamic health bar and user database.
* **Achievements:** Unlock custom emblems as you progress.
* **Retro Aesthetic:** Styled with pixelated graphic interfaces for a classic feel.

## 🚀 Getting Started

### Prerequisites
Ensure you have Python installed. You will also need the following libraries:
